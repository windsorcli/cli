# windsor configure

```sh
windsor configure
```

Configure workstation host/guest resources. Currently supports networking and DNS via the 'network' subcommand.

## Subcommands

- [`windsor configure network`](/reference/cli/commands/configure-network) — Configure workstation host/guest networking and DNS.
